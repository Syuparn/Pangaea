# Pangaea Playground

Try Pangaea online!: https://syuparn.github.io/Pangaea/

# About

Pangaea Playground is web Pangaea Runtime powered by Github Pages.
You can run Pangaea script by WebAssmbly Pangaea binary (see `/web/wasm`).
